Revisando conteudos

Conteudo do Primeiro Curso

HTML é uma "Linguagem" de marcação onde cada tipo de tag possui uma função específica.

Sempre que quiser lembrar dos conteúdos ou de tags, leia o conteúdo aqui que provavelmente você irá refrescar sua mente.

  • A tag <br> serve para inserir uma quebra de linha (novo parágrafo visualmente).
  • As tags <h1> criam o título principal, <h2> criam subtítulos relacionados ao <h1>, <h3> subtítulos de <h2>, e assim por diante, seguindo uma hierarquia de importância.
  • Usando uma série de caracteres especiais (entidades HTML), podemos adicionar emojis ou símbolos como "centavo": &cent;
  • Usando a tag <img> podemos adicionar imagens ao site através do atributo 'src' (source - caminho da imagem).
  • Curiosidade: Algumas tags HTML se tornam obsoletas com o tempo, por isso é sempre bom estar atualizado sobre as novas práticas e padrões.
  • Para adicionar negrito a uma palavra, podemos usar a tag <b> visto aqui, mas ela não possui significado semântico (não indica importância).
  • Mas usando a tag <strong> já possui um sentido semântico, indicando forte ênfase.
  • Usando a tag <i> sua frase fica em itálico, mas não tem significado semântico (apenas estilização).
  • Usando a tag <em> que agora sim possui semântica, indicando ênfase.
  • "Envelopamento" é um recurso extremamente útil para adicionar tags a partes específicas do texto rapidamente. No Visual Studio Code, por exemplo, pode ser feito selecionando o texto e usando Ctrl+Shift+P e digitando "Wrap with Abbreviation".
  • Podemos colocar marcas em textos usando a tag <mark>.
  • Para alterar o tamanho da fonte (embora geralmente se use CSS para isso), existiam as tags <big>big</big> e <small>small</small>.
  • Usando a tag <code> formatamos trechos de código inline, e usando a tag <pre> preservamos a formatação original do código (espaços e quebras de linha).
  • Criando uma página extra, você pode criar um link para ir para outro site ou página local usando a tag <a> (âncora). O atributo 'target="_self"' faz com que o link abra na mesma aba.
  • Podemos também adicionar sons com a tag <audio>.
